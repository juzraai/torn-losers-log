## Developer notes

**TLL** is a Nuxt project with `static` target.

```
# Setup
yarn

# Start development server
yarn run serve

# Generate site to "dist/"
yarn run build

# Release (bump version & push)
yarn run release && git push --follow-tags origin main
```
